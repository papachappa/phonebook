Phonebook app

Application was written on Python language, consist of two modules.
One is called "phonebook.py" contains internal simple phonebook class
architecture, including methods for creation phonebook,
adding/deleting/listing/searching records. Another is called app.py, is an entry
point to the app. App uses "cmd" module to emulate user inputs.

PREREQUISITES
Ubuntu 16.04 or higher
python3.5 or higher installed

In order to execute the app you must go to the folder where it is located and
type "python3 app.py" in termianl then follow the instructions or type help.
Type "CTRL+C" to exit

In order to execute unit tests you must go to unit_tests subfolder and type
"python3 -m unittest test_add_record.py"
